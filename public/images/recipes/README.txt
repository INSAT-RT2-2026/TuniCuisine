Place recipe images in this folder using the recipe slug as the filename.

Examples:
  mloukhia.jpg       → Mloukhia
  couscous-royal.jpg → Couscous Royal
  brik-a-l-oeuf.jpg  → Brik à l'Oeuf
  lablabi.jpg        → Lablabi
  makroudh.jpg       → Makroudh

Supported extensions: jpg, jpeg, png, webp

If no file is found, the site uses a default banner image.
