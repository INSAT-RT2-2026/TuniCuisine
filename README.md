## 📖 About The Project

**TuniCuisine** is a comprehensive web platform dedicated to showcasing the authentic flavors and culinary traditions of Tunisia. From the spicy delights of the Sahel to the hearty dishes of the interior, our platform celebrates Tunisia's diverse gastronomic heritage.

### 🎯 Mission

To preserve and share Tunisian culinary heritage with the world, making traditional recipes accessible to everyone while maintaining their authentic essence.

### ✨ What Makes Us Special

- 🗺️ **Regional Diversity**: Explore authentic dishes from all 24 governorates of Tunisia
- 📜 **Traditional Recipes**: Step-by-step instructions for classic Tunisian meals
- 🌶️ **Ingredient Guide**: Detailed information about traditional Tunisian ingredients
- 🤖 **AI Chatbot Assistant**: Get cooking tips and recipe recommendations
- 🎨 **Tunisian Design**: Beautiful interface inspired by traditional Tunisian art and culture
- 📱 **Responsive**: Seamless experience across all devices

## 🎨 Tunisian Design Elements

Our platform incorporates authentic Tunisian cultural elements:

### 🧢 Traditional Motifs
- **Chachia** 🧢 - Traditional Tunisian hat imagery in navigation
- **Zellige Tiles** 🔷 - Intricate geometric patterns in borders
- **Calligraphy** ✍️ - Arabic and French typography
- **Jasmine** 🌸 - National flower decorative elements
- **Sidi Bou Said Blue** 💙 - Iconic blue and white color scheme
- **Berber Patterns** 🔶 - Traditional geometric designs
- **Carthage Motifs** 🏛️ - Historical architectural elements

### 🎨 Color Palette
- **Tunisian Red** (#E70013) - Flag red
- **Mediterranean Blue** (#0072C6) - Sea and sky
- **Jasmine White** (#FFFFFF) - Purity and tradition
- **Harissa Orange** (#FF6B35) - Spicy warmth
- **Olive Green** (#708238) - Agricultural heritage
- **Desert Sand** (#EDC9A5) - Saharan landscape

## 🚀 Getting Started

### Prerequisites

```bash
# Node.js (v14 or higher)
node --version

# npm or yarn
npm --version
```

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/INSAT-RT2-2026/TuniCuisine.git
cd TuniCuisine
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
```

3. **Set up environment variables**
```bash
cp .env.example .env
# Edit .env with your configuration
```

4. **Run the development server**
```bash
npm start
# or
yarn start
```

5. **Open your browser**
```
Navigate to http://localhost:3000
```

## 🤝 Contributing

We welcome contributions from the community! Whether you're a developer, designer, chef, or food enthusiast, there's a place for you in TuniCuisine.

### How to Contribute

1. **Fork the repository**
2. **Create your feature branch**
   ```bash
   git checkout -b feature/AmazingFeature
   ```
3. **Commit your changes**
   ```bash
   git commit -m 'Add some AmazingFeature'
   ```
4. **Push to the branch**
   ```bash
   git push origin feature/AmazingFeature
   ```
5. **Open a Pull Request**

### Contribution Ideas

- 📝 Add new recipes with photos
- 🎨 Improve UI/UX design
- 🌍 Translate content to other languages
- 🐛 Fix bugs and issues
- 📖 Improve documentation
- 🧪 Write tests
- 💡 Suggest new features

### Recipe Submission Guidelines

When adding recipes, please include:
- Recipe name in Arabic and French/English
- High-quality photo (minimum 1200x800px)
- Complete ingredient list with measurements
- Step-by-step instructions
- Cooking time and difficulty level
- Regional origin
- Cultural context or history (optional)
